# Inveon Course App
Inveon Course App is a course sales website project.

## Tech Stack
### Backend
- .NET 8
- PostgreSql
- Entity Framework Core
- Jwt
- MassTransit - RabbitMQ
- Redis
- Cloudflare R2 Object Storage
- Iyzipay
- Gmail SMTP
- Serilog
- FluentValidation
  
### Frontend
- React
- TailwindCSS
- ReactIcons
- ReactToastify
- Axios
